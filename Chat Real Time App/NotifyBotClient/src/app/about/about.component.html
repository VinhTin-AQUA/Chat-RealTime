Khái quát về ứng dụng

Back-end

  • - Asp Net Core API
  • - Sử dụng Identity để quản lý người dùng, xác thực quyền truy cập.
  • - SignalR để xây dựng chức năng chat-realtime

Front-end

  • - Angular
  • - Sử dụng SignalR để giao tiếp với Hub API.

Nếu bạn quan tâm đến ứng dụng, hãy liên hệ cho chúng tôi để trao đổi trực tiếp hoặc tham khảo mã nguồn bên dưới.
